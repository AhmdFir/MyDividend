## ‚ú® Features
This application meets the following assignment requirements:

* **Input Capability:**
  * [cite_start]Accepts **Invested Fund Amount** (RM)[cite: 7].
  * [cite_start]Accepts **Annual Dividend Rate** (%)[cite: 7].
  * [cite_start]Accepts **Number of Months Invested** (with validation to ensure a maximum of 12 months)[cite: 7].
* **Smart Calculation:**
  * [cite_start]Calculates **Monthly Dividend** using the formula: `(Rate / 12) * Invested Fund`[cite: 9].
  * [cite_start]Calculates **Total Dividend** using the formula: `Monthly Dividend * Number of Months`[cite: 10].
* **Clear Output:**
  * [cite_start]Displays the final result formatted clearly to **two decimal places** (e.g., RM 2500.00)[cite: 11].
* **Navigation & UI:**
  * [cite_start]Features a responsive "Mobile First" design[cite: 24].
  * [cite_start]Includes a navigation menu (Home & About)[cite: 14].
  * [cite_start]**About Page** contains the app icon, author details, copyright notice, and a clickable link to this GitHub repository[cite: 17, 21].
